**TYCHON VALIDATION RULES:**

Every test must:
1. **Force the failure** (don't wait for it)
2. **Verify immediate exposure** (exception thrown, not logged)
3. **Prove no silent continuation** (operation stops, not proceeds)
4. **Check propagation** (error reaches caller)

**TEST NAMING:**
- `[Fact] public async Task [Component]_[Failure]_[ExpectedBehavior]`
- Example: `SignalGenerator_Underfill_ReturnsFalseNotTrue`

**ASSERTION PATTERNS:**
```csharp
// Good - proves failure is exposed
await Assert.ThrowsAsync<InvalidOperationException>(() => operation);

// Good - proves partial failure detected
var result = await operation;
Assert.False(result.Success);
Assert.Equal(50, result.Inserted);
Assert.Equal(100, result.Requested);

// Bad - proves nothing
var result = await operation;
Assert.NotNull(result); // Could be lying
```
