## Key Learnings

### Timestamps & Cron
- **Critical lesson:** Never guess timestamps
- Must get current time from session_status before calculating
- Cron "at" jobs appear to have issues (investigation needed)
- Pivoted to heartbeat + JSON reminders instead

### Model Switching
- Per-session overrides work perfectly (no config pollution)
- No need to patch config or restart gateway
- Much cleaner than global config changes
