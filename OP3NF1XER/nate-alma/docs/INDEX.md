# Nate-Alma Documentation Index

This folder contains the full operator documentation for maintaining Nate's
OpenClaw deployment through the SSH control plane.

- `SETUP_AND_COMMANDS.md` - setup, prerequisites, and daily command reference.
- `DEPLOYED_CONFIG_TOKEN_PLUG_POINTS.md` - where to plug Claude setup-token and
  Telegram bot token in deployed config paths.
- `OPERATIONS_RUNBOOK.md` - detect/diagnose/recover/verify workflow.
- `FAILURE_TRACEABILITY.md` - failure points, risk markers, and log strategy.
- `SESSION_CONTINUITY.md` - resume protocol for new sessions and handoffs.
- `DECISION_142_GOVERNANCE.md` - governance contract, parity matrix, closure
  rules, and deliverable policy.

Primary control scripts are in `../tools`.
